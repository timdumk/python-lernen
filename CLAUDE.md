# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is a personal Python learning project ("python-lernen" = "learning Python" in German). All concepts are practiced in a single file, `main.py`, added incrementally as new topics are covered.

## Running the code

```bash
python3 main.py
```

Note: `main.py` uses `input()` calls, so it will pause and wait for keyboard input when run.

## Structure

Everything lives in `main.py`. Topics covered so far (in order of introduction):
- Conditionals (`if`/`elif`/`else`)
- Loops (`for`, `while`)
- Functions (definition, calls, `return`)
- Lists (`append`, `remove`, indexing)
- Dictionaries
- `try`/`except`
- `import` and the `random` module
